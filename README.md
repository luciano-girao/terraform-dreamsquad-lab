# Terraform DreamSquad Lab

Laboratório de Terraform usado no teste da DreamSquad, com foco em infraestrutura como código para provisionamento básico em AWS.

## Objetivo

Demonstrar uso prático de Terraform para criar e gerenciar recursos de nuvem com uma abordagem organizada, reutilizável e versionada.

## O que este projeto inclui

- Estrutura inicial de Terraform
- Definição de recursos básicos em AWS
- Variáveis e outputs
- Exemplo de organização para laboratório e estudos

## Como usar

1. Instale o Terraform
2. Configure suas credenciais da AWS
3. Execute:

```bash
terraform init
terraform plan
terraform apply
```

## Observação

Este repositório foi criado para fins de estudo e apresentação do uso de Terraform em cenários de cloud.
